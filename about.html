About Twee-Fi

Twee-Fi is an Open Source project initiated by FactsMission AG to review the claims made in Twitter. The name Twee-Fi stands for "Tweet Fidelity".

Twee-Fi uses Linked Data technologies, the schema.org vocabulary, wich is supported by major search engines and is built on Solid technologies to allow for a decentralized web with data controlled by the users.

FactsMission is a Swiss Company founded in 2017 with the goal to use Linked Data technologies to promote the ability to identify facts and to fight fake news on the web. FactsMission is a member of the W3C.
